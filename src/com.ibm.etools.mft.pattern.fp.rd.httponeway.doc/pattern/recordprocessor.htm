RecordProcessor subflow for the Record Distribution to HTTP: one-way pattern

Details about the RecordProcessor subflow for the Record Distribution to HTTP: one-way pattern.

By default, this subflow has no impact. When the Record Distribution to HTTP: one-way pattern is used to generate a pattern instance, the RecordProcessor subflow contains only a Passthrough node. This subflow is included to provide a place where a pattern instance can be customized by using standard tools, without changing the structure of the pattern.
